## 📋 Overview

This repository documents the design and deployment of a **complete enterprise IT infrastructure** for StarTech Business College.

The project simulates a real-world organizational environment integrating:

- 🖥️ Endpoint systems (students, staff, admin)
- 🗄️ Server infrastructure
- 🌐 Network architecture (LAN + Wi-Fi)
- 🔐 Identity & Access Management (Active Directory)
- ☁️ Microsoft 365 cloud integration
- 🛡️ Security and access control mechanisms

## 🧪 Implementation Summary

### 🌐 Network Architecture
- Designed structured campus network
- Implemented segmentation for departments
- Configured switches, router, and access points

---

### 🖥️ Infrastructure Setup
- Deployed **servers (HPE DL380 & DL360)**
- Configured endpoint systems (students & staff)
- Installed printers, projectors, and CCTV systems

---

### 🔐 Identity & Access Management
- Created **Active Directory domain**
- Designed Organizational Units (OUs):
  - Admin Team
  - Teachers
  - Marketing
  - Accounting
  - Management
  - Reception
- Implemented **Role-Based Access Control (RBAC)**

---

### ☁️ Microsoft 365 Integration
- Connected custom domain
- Configured user accounts and licenses
- Enabled cloud-based identity and collaboration

---

### 🛡️ Security Implementation
- Fortinet Firewall
- Applied access control policies
- Secured network segmentation

---

### ✅ System Validation
- Verified domain authentication
- Tested user login across departments
- Confirmed Microsoft 365 functionality
- Ensured network connectivity and security policies
